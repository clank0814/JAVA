GIT - GITHUB
git init --> Permite inicializar repositorio local desde una carpeta o archivo.
git status ---> Visualizar cambios en el repositorio (añadido desde la inicilizacion) para realizar summit (guardado)
git remote (add origin) ---> Agregar repositorio remoto de github
git remote -v ---> (visulizacion de vinculacion con github)
git commit -m "etiqueta" ---> Guardado de desarrollo
git diff ---> diferencias entre el commit anterior y desarrollo actual
git pull (origin...) ---> Inicializa desde github el desarrollo y contenido de proyectos ya trabajados 
git push (origin...) ---> Sube a github el desarrollo y contenido de proyectos en los que se trabaja de manera local.
git checkout (nombre_de_rama) ---> Permite saltar entre ramas y versiones. Tambien se utiliza para descartarlas.
git checkout (etiqueta_de_commit) ---> Permite regresar a un commit anterior.
git log ---> Permite ver el historial de trabajo.
git --help ---> Explica cada uno de los comandos disponibles en git

JAVA CON NETBEANS (ATAJOS):
Ctrl. + E : Borra línea de codigo donde estoy posicionado.
Ctrl. + S : Guarda los cambios en la clase que estoy posicionado.
Shift. + F6 : RUN main proyect (actual).
Alt. + Shift + F : Le da formato al código.(queda mas legible, ordenado)
Alt + Enter : Se usa sobre el texto subrayado para ver la sugerencia que nos hace el IDE cuando hay
algún error de sintaxis o se necesita realizar algún import o surround.
Ctrl. + Shift + : Hace una copia de la línea de código que estabas posicionado.
Shift + DELETE : Borra toda la línea de código donde estoy posicionada.
Ctrl. + Space : Completa el código que estamos escribiendo.
Ctrl. + U U : Convierte el código a mayúsculas.
Ctrl. + U L : Convierte el código a minúsculas.
Alt + Insert : Agrega constructores, métodos accesotes, propiedades.(codigos predeterminados)
Ctrl. + G : Ir numero de línea concreto(introducida x ventana auxiliar).
Ctrl. +[NOMBRECLASE] : Me despliega todas las características de la clase.
